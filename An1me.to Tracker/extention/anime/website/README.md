# An1me.to Tracker Website

Professional landing page for the An1me.to Tracker Chrome Extension.

## Features

- Modern dark theme with anime aesthetics
- Dynamic version fetching from Chrome Web Store
- Fully responsive design
- GDPR-compliant Privacy Policy & Terms of Service
- Optimized for Google OAuth verification

## Quick Start

1. Copy screenshots to assets folder
2. Deploy to GitHub Pages
3. Website will be live at `https://YOUR_USERNAME.github.io/REPO_NAME/`

## Deployment

### GitHub Pages

```bash
cd D:\Projects\anime
git add website/
git commit -m "Add website"
git push origin main
```

Enable GitHub Pages:
- Settings → Pages
- Source: main branch
- Folder: /website
- Save

## File Structure
