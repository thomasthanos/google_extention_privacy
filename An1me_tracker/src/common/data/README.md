# 📊 common/data/ — Δεδομένα & επεξεργασία δεδομένων

«Έξυπνοι» βοηθοί που δουλεύουν πάνω στα δεδομένα σου (anime, πρόοδος, slugs), κοινοί για όλο το extension.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **anilist-core.js** | Ο **πυρήνας επικοινωνίας με το AniList** (ερωτήματα GraphQL, cache, σύνδεση). Τον χρησιμοποιούν και το popup και το background. |
| **merge-utils.js** | Η λογική που **ενώνει** τα τοπικά σου δεδομένα με αυτά του cloud **χωρίς να χάνεται τίποτα** (ένωση anime, προόδου, εξάλειψη διπλότυπων, διόρθωση «ψεύτικων» διαρκειών). |
| **slug-migration.js** | **Μετονομάζει παλιά slugs σε νέα/σωστά** όταν ο ιστότοπος αλλάζει διευθύνσεις — ώστε να μη «χαθεί» η πρόοδός σου. |
| **multipart-mappings.js** | Μικρός πίνακας: **ενώνει anime που ο ιστότοπος έχει σπασμένα** σε πολλά κομμάτια (π.χ. Bleach, Fate/Zero) και διορθώνει την αρίθμηση επεισοδίων. |

> 💡 **slug** = το αναγνωριστικό ενός anime στη διεύθυνση (π.χ. `fate-zero`).
