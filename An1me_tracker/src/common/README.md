# 🤝 common/ — Κοινός κώδικας για όλους

Εδώ μένει κώδικας & δεδομένα που τα **μοιράζονται και τα 3 μέρη** του extension (background, content, popup) — για να μην γράφουμε το ίδιο πράγμα 3 φορές.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **cloud.js** | Όλη η «υδραυλική» του cloud & της σύνδεσης: ρυθμίσεις Firebase, αναγνώριση/ταξινόμηση σύνδεσης, tokens, και μετατροπή των δεδομένων σου σε μορφή που καταλαβαίνει το Firestore (και πίσω). |
| **logger.js** | Το σύστημα **καταγραφής (logs)**: όμορφα, χρωματιστά μηνύματα στην κονσόλα του προγραμματιστή, με «φρένο» ώστε να μη γεμίζει με επαναλήψεις. |

Υποφάκελος:
- **data/** — δεδομένα & «έξυπνοι» βοηθοί για AniList, ένωση (merge) δεδομένων και slugs. *(δες το README μέσα)*
