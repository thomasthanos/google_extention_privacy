# 🃏 popup/cards/ — Η εμφάνιση των καρτών anime

Εδώ φτιάχνεται κάθε **«κάρτα» anime** που βλέπεις στη λίστα. Τα αρχεία `-inprogress`, `-movies`, `-seasons` **προσθέτουν** λειτουργίες στον **ίδιο** renderer (`AnimeCardRenderer`) — απλώς χωρισμένα ανά τύπο για ευκολία.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **anime-card.js** | Ο **βασικός** renderer μιας κανονικής κάρτας anime (πρόοδος, επεισόδια, μετρήσεις canon/filler). |
| **anime-card-inprogress.js** | Προσθέτει την εμφάνιση για τα **«αυτά που βλέπεις τώρα»**. |
| **anime-card-movies.js** | Προσθέτει την εμφάνιση για **ταινίες**. |
| **anime-card-seasons.js** | Προσθέτει την **ομαδοποίηση πολλών σεζόν** σε μία ενιαία κάρτα. |
| **share-card.js** | Φτιάχνει μια **εικόνα που μπορείς να μοιραστείς** (π.χ. social), με τα στατιστικά/πρόοδό σου ζωγραφισμένα πάνω της. |
