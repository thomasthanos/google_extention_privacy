# 🖥️ popup/views/ — Οι μεγάλες οθόνες

Οι «πλήρεις οθόνες» του popup — **και η εμφάνισή τους και τα μαθηματικά** πίσω τους. Γενικός κανόνας: τα `-view` = η **εμφάνιση**, τα `-engine` = οι **υπολογισμοί**.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **stats-view.js** | Η οθόνη **«Στατιστικά»** (γραφήματα & νούμερα). |
| **stats-engine.js** | Τα **μαθηματικά** των στατιστικών (υπολογισμοί ανά μέρα / εβδομάδα / μήνα). |
| **goals-view.js** | Η οθόνη **«Στόχοι»** (ημερήσιοι / εβδομαδιαίοι / μηνιαίοι στόχοι, επίπεδα). |
| **achievements-engine.js** | Η **λογική για τα achievements/badges** — ποια έχεις ξεκλειδώσει και πότε. |
| **settings-view.js** | Η οθόνη **«Ρυθμίσεις»**. |
