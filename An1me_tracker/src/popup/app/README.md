# 🧩 popup/app/ — Τα κύρια κομμάτια λειτουργιών

Εδώ είναι τα μεγάλα «κομμάτια» του popup, **βγαλμένα από το `main.js`** για να είναι πιο καθαρά & συντηρήσιμα. Το καθένα κάνει **μία δουλειά**.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **add-anime-dialog.js** | Το παράθυρο **«Προσθήκη / Επεξεργασία anime»** (ανίχνευση slug, προεπισκόπηση επεισοδίων, fillers, αλλαγή τίτλου). |
| **anime-actions.js** | Οι **ενέργειες** πάνω σε ένα anime: διαγραφή, σήμανση completed / dropped / favorite / on-hold, καθάρισμα όλων. |
| **auth-ui.js** | Η **σύνδεση**: Google + email/κωδικός + «ξέχασα τον κωδικό». |
| **render-list.js** | **Φτιάχνει τη λίστα** anime: ταξινόμηση, ομαδοποίηση, «μοντάρισμα» των καρτών στην οθόνη. |
| **stats-views.js** | Η **μπάρα στατιστικών** + η εναλλαγή ανάμεσα στις οθόνες (stats / goals / settings). |
| **metadata-repair.js** | Το **UI κατάστασης** για τη διόρθωση metadata + η ροή «κατέβασε όλα τα fillers». |
| **set-password-modal.js** | Το παράθυρο **«όρισε κωδικό»**. |
| **toasts.js** | Τα μικρά **μηνύματα επιβεβαίωσης/λάθους** (toasts) που εμφανίζονται για λίγο. |

> 🔧 Μοιράζονται κατάσταση μέσω `AT.PopupState` και δέχονται «εξαρτήσεις» από το `main.js` μέσω `_init()`.
