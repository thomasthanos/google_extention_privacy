# 🪟 popup/ — Το παραθυράκι του extension

Αυτό είναι το **popup** (και το side panel) — ό,τι ανοίγει όταν πατάς το **εικονίδιο** του extension. Εδώ βλέπεις τη λίστα σου, τα στατιστικά, τις ρυθμίσεις κ.λπ.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **main.js** | Το **αρχείο-αφετηρία** που «δένει» όλα τα παρακάτω μαζί (orchestrator). Ξεκινά το popup και συντονίζει τι εμφανίζεται. *(Περιέχει επίσης το donate dropdown και το backup/εξαγωγή βιβλιοθήκης.)* |

Υποφάκελοι:

| Υποφάκελος | Τι κάνει |
|---|---|
| **app/** | Τα κύρια **κομμάτια λειτουργιών** (παράθυρα, ενέργειες, σύνδεση, render λίστας…). |
| **cards/** | Η **εμφάνιση των «καρτών»** anime στη λίστα. |
| **lib/** | **Βοηθοί & εργαλεία** για όλο το popup. |
| **services/** | Επικοινωνία με **εξωτερικές υπηρεσίες** (AniList, Firebase, fillers). |
| **views/** | Οι μεγάλες **«οθόνες»**: Στατιστικά, Στόχοι, Ρυθμίσεις, Achievements. |

*(δες το README μέσα σε κάθε υποφάκελο)*

> 🔧 **Πώς «μιλάνε» μεταξύ τους:** όλα τα κομμάτια κρεμιούνται στο `window.AnimeTracker` (συντομογραφία `AT`). Τα modules του `app/` μοιράζονται κατάσταση μέσω `AT.PopupState` και «δένονται» από το `main.js` με μια συνάρτηση `_init()`.
