# 🧰 popup/lib/ — Εργαλεία & βοηθοί του popup

Μικρά, επαναχρησιμοποιήσιμα εργαλεία που χρησιμοποιεί **όλο** το popup.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **ui-helpers.js** | Βοηθοί **εμφάνισης**: μορφοποίηση ωρών/ημερομηνιών, εικονίδια, ασφαλές κείμενο (escape). Περιέχει επίσης το **Logger** και τα **Dialogs** (παράθυρα + προσβασιμότητα). |
| **storage.js** | **Διάβασμα/γράψιμο** της αποθήκευσης του popup. Περιέχει και το **SlugUtils** (καθαρισμός/κανονικοποίηση slug). |
| **config.js** | **Ρυθμίσεις & νούμερα** του popup (διάρκειες cache, καθυστερήσεις, όρια εμφάνισης). |
| **anime-status.js** | Η λογική **κατάστασης** ενός anime: watching / completed / airing / dropped / on-hold. |
| **progress-manager.js** | **Διαχείριση προόδου**: κανονικά slugs, αφαίρεση διπλότυπων επεισοδίων, καθάρισμα. |
| **maintenance.js** | **«Συντήρηση» δεδομένων**: διορθώνει διάρκειες ταινιών, σβήνει «φαντάσματα», καθαρίζει ημερομηνίες από import. |
| **filler-fetch-ui.js** | Το **overlay προόδου** που βλέπεις όσο κατεβαίνουν δεδομένα filler. |
| **episode-parse.js** | Μετατρέπει κείμενο σαν **`1-12, 15`** σε λίστα επεισοδίων (για το παράθυρο προσθήκης). |
