# 🗺️ src/ — Χάρτης κώδικα

Εδώ ζει όλος ο κώδικας του extension. Χωρίζεται σε **4 «κόσμους»** — ο καθένας τρέχει σε διαφορετικό σημείο:

| Φάκελος | Τι είναι | Σε απλά λόγια |
|---|---|---|
| **background/** | Service worker | Ο «αόρατος εργάτης» που τρέχει στο παρασκήνιο, ακόμη κι όταν δεν έχεις ανοιχτό το an1me. Κατεβάζει δεδομένα, στέλνει ειδοποιήσεις, συγχρονίζει. |
| **content/** | Content scripts | Ο κώδικας που «μπαίνει μέσα» στις σελίδες του an1me.to και δουλεύει εκεί (π.χ. όσο βλέπεις επεισόδιο). |
| **popup/** | Popup & side panel | Το παραθυράκι που ανοίγει όταν πατάς το εικονίδιο του extension. |
| **common/** | Κοινός κώδικας | Κώδικας & δεδομένα που τα μοιράζονται και οι 3 παραπάνω. |

Επίσης: **icons/** (εικονίδια) και **fonts/** (γραμματοσειρές) — απλά assets, όχι κώδικας.

> 📌 Το αρχείο-αφετηρία του background είναι το **`background.js`** στη ρίζα του project (όχι μέσα στο `src/`).

Κάθε υποφάκελος έχει το δικό του `README.md` που εξηγεί αρχείο-αρχείο τι κάνει. 🙂

**Μικρό λεξικό:**
- **slug** = το «όνομα» ενός anime μέσα στη διεύθυνση, π.χ. `one-piece` στο `an1me.to/watch/one-piece`.
- **filler** = επεισόδια «γεμίσματος» που δεν ανήκουν στην κανονική ιστορία (canon).
- **sync** = συγχρονισμός (να ταιριάζουν τα δεδομένα παντού: τοπικά, cloud, AniList).
