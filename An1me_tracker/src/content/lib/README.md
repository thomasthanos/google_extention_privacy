# 🧰 content/lib/ — Βοηθοί για τις σελίδες

Μικρά εργαλεία που χρησιμοποιεί ο κώδικας μέσα στις σελίδες του an1me.to.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **config.js** | **Ρυθμίσεις & νούμερα** για την παρακολούθηση: κάθε πότε σώζει την πρόοδο, πόσα δευτερόλεπτα χρειάζονται για να μετρήσει ένα επεισόδιο, σε τι ποσοστό θεωρείται «completed» κ.λπ. |
| **storage.js** | **Διάβασμα & γράψιμο** στην αποθήκευση του Chrome, από μέσα από τις σελίδες. |
| **notifications.js** | Τα μικρά **μηνύματα/ειδοποιήσεις** που εμφανίζονται πάνω στη σελίδα (+ φόρτωση γραμματοσειράς, καθάρισμα όταn φεύγεις). |
