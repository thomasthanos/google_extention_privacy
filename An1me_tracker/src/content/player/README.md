# ▶️ content/player/ — Γύρω από τον video player

Όλα όσα συμβαίνουν **όσο παίζει το βίντεο** ενός επεισοδίου.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **video-monitor.js** | **Παρακολουθεί το βίντεο**: πότε παίζει, πότε σταματά, πότε αλλάζει server· ξεκινά τα χρονόμετρα που σώζουν την πρόοδο. |
| **progress-tracker.js** | Κρατάει την **πρόοδό σου** και αποφασίζει **πότε να τη σώσει** και **πότε ένα επεισόδιο μετράει ως «ολοκληρωμένο»**. |
| **episode-writer.js** | **Γράφει** τα δεδομένα προόδου ενός επεισοδίου στην αποθήκευση (διάρκειες, χρόνοι). |
| **episode-highlight.js** | **Χρωματίζει** στη λίστα επεισοδίων ποια έχεις ήδη δει και ποια είναι filler. |
| **skiptime-helper.js** | Το κουμπί **«Skip Intro / Skip Outro»** — η κύρια λογική. |
| **skiptime-utils.js** | Καθαροί βοηθοί + ρυθμίσεις για το skip (χωρίς κατάσταση/state). |
| **skiptime-styles.js** | Το **CSS (στυλ)** του κουμπιού skip. |
