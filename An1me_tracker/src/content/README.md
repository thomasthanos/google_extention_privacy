# 🌐 content/ — Κώδικας μέσα στις σελίδες του an1me.to

Αυτά τα αρχεία «**μπαίνουν μέσα**» στις σελίδες του an1me.to (content scripts) και δουλεύουν εκεί — κυρίως όσο βλέπεις ένα επεισόδιο.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **main.js** | Ο **«εγκέφαλος» της σελίδας παρακολούθησης** (`/watch/…`). Συντονίζει τα πάντα όσο βλέπεις επεισόδιο: ξεκινά/σταματά την παρακολούθηση, βρίσκει ποιο επεισόδιο παίζει, διαχειρίζεται την αλλαγή server, σώζει την πρόοδο. |

Υποφάκελοι:

| Υποφάκελος | Τι κάνει |
|---|---|
| **lib/** | Μικροί βοηθοί: ρυθμίσεις, αποθήκευση, ειδοποιήσεις πάνω στη σελίδα. |
| **page/** | Κώδικας για συγκεκριμένες σελίδες: αρχική («Continue Watching»), parsing, watchlist, αποτροπή αντιγραφής. |
| **player/** | Όλα γύρω από τον **player**: πρόοδος βίντεο, skip intro/outro, highlight επεισοδίων. |

*(δες το README μέσα σε κάθε υποφάκελο)*
