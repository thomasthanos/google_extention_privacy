# 📄 content/page/ — Κώδικας ανά σελίδα

Κώδικας για **συγκεκριμένες σελίδες** του an1me.to (όχι τον ίδιο τον player).

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **anime-parser.js** | Βγάζει τις **πληροφορίες του anime** (slug, τίτλος, αριθμός επεισοδίου) από τη διεύθυνση και τη σελίδα. |
| **watchlist-sync.js** | Συγχρονίζει την **κατάστασή σου** (Watching / Completed κ.λπ.) με τη watchlist του **ίδιου του ιστότοπου**. |
| **continue-watching.js** | Φτιάχνει το ράφι **«Continue Watching»** στην αρχική σελίδα (αυτά που βλέπεις τώρα). |
| **continue-watching-utils.js** | Καθαροί βοηθοί για το παραπάνω — μετατρέπουν την αποθηκευμένη πρόοδο σε στοιχεία για την οθόνη (χωρίς να αγγίζουν τη σελίδα). |
| **continue-watching-styles.js** | Το **CSS (στυλ/εμφάνιση)** του ραφιού «Continue Watching». |
| **copy-guard.js** | **Εμποδίζει την αντιγραφή κειμένου** στον ιστότοπο (με κάποιες εξαιρέσεις). Τρέχει πολύ νωρίς, σε όλες τις σελίδες. |
