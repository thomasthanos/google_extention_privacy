# ⏰ background/jobs/ — Εργασίες που τρέχουν μόνες τους

«Jobs» = εργασίες που ξυπνούν **μόνες τους** σε τακτά διαστήματα (με χρονοδιακόπτες/alarms του Chrome) και κάνουν μια δουλειά στο παρασκήνιο.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **metadata-repair.js** | Σιγά-σιγά **φρεσκάρει & διορθώνει** τα στοιχεία των anime σου (σύνολο επεισοδίων, τύποι επεισοδίων, εικόνες). Δουλεύει λίγα-λίγα (μερικά τη φορά) για να μην επιβαρύνει. |
| **smart-notifications.js** | Ελέγχει περίπου **κάθε 1 ώρα** αν βγήκε **νέο επεισόδιο** σε anime που παρακολουθείς, και σου στέλνει **ειδοποίηση**. |
