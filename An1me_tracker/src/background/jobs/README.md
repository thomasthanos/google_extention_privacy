# ⏰ background/jobs/ — Εργασίες που τρέχουν μόνες τους

«Jobs» = εργασίες που ξυπνούν **μόνες τους** σε τακτά διαστήματα (με χρονοδιακόπτες/alarms του Chrome) και κάνουν μια δουλειά στο παρασκήνιο.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **metadata-repair.js** | Σιγά-σιγά **φρεσκάρει & διορθώνει** τα στοιχεία των anime σου (σύνολο επεισοδίων, τύποι επεισοδίων, εικόνες). Δουλεύει λίγα-λίγα (μερικά τη φορά) για να μην επιβαρύνει. |
| **smart-notifications.js** | Ξυπνάει περίπου **κάθε 1 ώρα**, αλλά είναι **έξυπνο**: ελέγχει κάθε anime **γύρω από την ώρα που πέφτει το επόμενο επεισόδιο** (`nextEpisodeAt`) αντί για όλα μαζί. Αν δεν έχει προγραμματισμό, πέφτει πίσω σε χαλαρό ρυθμό. Βάζει προτεραιότητα στα πιο επείγοντα, δεν ξαναειδοποιεί για το ίδιο επεισόδιο, και αν βγουν **πολλά μαζί** στέλνει **μία ομαδική** ειδοποίηση αντί για πολλές. |
