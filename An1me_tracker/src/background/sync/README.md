# 🔄 background/sync/ — Συγχρονισμός με άλλες υπηρεσίες

«Sync» = κρατούν την πρόοδό σου **ίδια παντού** — όχι μόνο μέσα στο extension, αλλά και σε εξωτερικές υπηρεσίες.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **anilist-sync.js** | Στέλνει την πρόοδό σου στο **AniList** (αν έχεις συνδέσει λογαριασμό AniList). Σπρώχνει τις αλλαγές στο παρασκήνιο, λίγα-λίγα. |
| **watchlist-sync.js** | Συγχρονίζει τη **watchlist** σου με τον **ίδιο τον ιστότοπο an1me.to** (π.χ. όταν βάζεις κάτι σε «Watching», ενημερώνεται και εκεί). |
