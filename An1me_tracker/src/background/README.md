# ⚙️ background/ — Ο αόρατος εργάτης

Αυτός είναι ο **service worker**: τρέχει στο παρασκήνιο του browser, **χωρίς** να φαίνεται κάπου. Κάνει τις βαριές & χρονοβόρες δουλειές ώστε το popup και η σελίδα να μένουν γρήγορα.

> Το αρχείο-αφετηρία είναι το **`background.js`** (στη ρίζα του project). Αυτό φορτώνει όλα τα παρακάτω και χειρίζεται τη σύνδεση με το Firebase/Firestore (cloud).

Χωρίζεται σε 3 υποφακέλους:

| Υποφάκελος | Τι κάνει |
|---|---|
| **fetchers/** | Κατεβάζουν δεδομένα από εξωτερικές πηγές (skip times, fillers, info anime). |
| **jobs/** | Προγραμματισμένες εργασίες που τρέχουν **μόνες τους** κάθε λίγη ώρα. |
| **sync/** | Κρατούν τα δεδομένα σου συγχρονισμένα με εξωτερικές υπηρεσίες (AniList, ο ίδιος ο ιστότοπος). |

Δες το `README.md` μέσα σε κάθε υποφάκελο για λεπτομέρειες. 🙂
