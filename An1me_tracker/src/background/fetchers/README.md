# 📥 background/fetchers/ — Κατεβάζουν δεδομένα

«Fetchers» = τα κομμάτια που **κατεβάζουν πληροφορίες από το ίντερνετ** (από άλλες υπηρεσίες/σελίδες) και τις αποθηκεύουν για να τις χρησιμοποιεί το extension.

| Αρχείο | Τι κάνει (απλά) |
|---|---|
| **aniskip.js** | Βρίσκει **πότε ξεκινά & τελειώνει το intro/outro** κάθε επεισοδίου (από την υπηρεσία AniSkip). Αυτό τροφοδοτεί το κουμπί «Skip Intro / Skip Outro». Κρατάει cache για να μην ξαναρωτάει συνέχεια. |
| **filler-discovery.js** | Βρίσκει **ποια επεισόδια είναι filler** (από το AnimeFillerList). Έχει μέσα μια λίστα με τα γνωστά anime που έχουν fillers (Naruto, One Piece, Bleach…). |
| **an1me-scraper.js** | «Διαβάζει» σελίδες του ίδιου του an1me.to για να βγάλει πληροφορίες anime — αντιστοιχίζει slugs (π.χ. σελίδα παρακολούθησης → σελίδα info), αναγνωρίζει σεζόν κ.λπ. |
